# bloop_ding

`bloop_ding` is the planned terminal notifier in the Bloop toolkit.

## Product intent

When an AI coding session is waiting for user input, play a short local sound so the user can come back to the terminal.

## Scope (v1)

- Local only
- Minimal setup
- Works with popular terminal workflows
- Optional custom sound
- Quiet mode / cooldown to avoid notification spam

## Suggested CLI

- `bloop-ding run`
- `bloop-ding test`
- `bloop-ding config`

## Naming

Recommended name: **Bloop Ding**

Alternates:
- Bloop Notify
- Bloop Ping
